## 📈 Como acompanhar seu progresso
Você pode criar o seu **Diário de Bordo** abrindo uma Issue automática. Lá, você terá todas as matérias em formato de checklist (`[ ]`) para ir marcando conforme avança.

1. Vá até a aba **[Issues](../../issues)** no topo deste repositório.
2. Clique no botão verde **"New Issue"**.
3. Escolha **"Iniciar meu Curso"**.
