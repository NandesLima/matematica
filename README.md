## 🎯 Sobre este Repositório

Este repositório contém a curadoria e organização para uma formação em **Matemática** de nível universitário, focada em autodidatas. O projeto utiliza o framework **MkDocs** para gerar uma interface de aprendizado limpa e moderna.

### 🛠️ Diferenciais
- [x] **MkDocs Integration:** Navegação fluida entre tópicos (Cálculo, Álgebra, Estatística).
- [x] **Design Neon Dark:** Estética padronizada com o ecossistema **Sekhel**.
- [x] **GitHub Actions:** CI/CD configurado para deploy automático no GitHub Pages.

---

## 📚 Conteúdo Programático

A trilha está organizada para levar o estudante do nível básico ao avançado:

1.  **Fundamentos:** Pré-Cálculo e Lógica Matemática.
2.  **Cálculo:** Diferencial, Integral e Multivariável.
3.  **Álgebra:** Linear e Estruturas Algébricas.
4.  **Estatística:** Probabilidade e Inferência.
5.  **Avançado:** Equações Diferenciais e Análise Real.
